# Spotify Clone

A frontend clone of the Spotify Web Player built using **HTML, CSS, and JavaScript**.  
This project recreates the **Spotify web experience** with dynamic playlists, music controls, and a responsive user interface.

## 📄 Overview

The objective of this project is to practice frontend development concepts by recreating a real-world music streaming application UI.

The project demonstrates:
- Responsive frontend layouts
- Dynamic playlist loading
- DOM manipulation with JavaScript
- Audio handling using the JavaScript Audio API
- Interactive music player controls

---

## ✨ Features

- 🎵 Spotify-inspired modern UI
- 📂 Dynamic playlist loading from folders
- ▶️ Play / Pause functionality
- ⏮️ Previous / Next track controls
- 🔊 Volume control and mute option
- ⏱️ Real-time song duration & seekbar
- 📱 Responsive sidebar navigation
- 🌙 Dark theme design

---

## 🛠️ Technologies Used

- HTML5
- CSS3 (Flexbox & Responsive Design)
- JavaScript (DOM Manipulation & Audio API)
- Node.js
- Express.js
